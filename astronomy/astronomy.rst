Python in Astronomy
===================


**Before you proceed**

Download `this <http://mpia.de/~robitaille/python/python4cardiff.tar.gz>`_ file and then
expand it and go into this directory::

    tar xvzf python4cardiff.tar.gz
    cd python4cardiff

Then start up IPython with the ``--pylab`` option to enable easy plotting::

    ipython --pylab
